## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

### 2. Navigate to the project folder

```bash
cd portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```
